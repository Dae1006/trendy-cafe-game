## How to Use

### Starting a Session
1. **Read TEAM.md** — team structure overview
2. **Read CEO.md** — orchestrator instructions
3. **Read specific agent files** as needed for that domain

### Agent Workflow
```
1. CMO researches trends → outputs to market_report_YYYY-MM-DD.md
2. CCO researches ops → outputs to ops_report_YYYY-MM-DD.md
3. CEO reviews reports → creates weekly_plan_YYYY-MM-DD.md
4. CTO implements priority features
5. CEO reviews CTO work → deploys to GitHub Pages
```

### Task Assignment Format
When assigning tasks, use this format:
```
## TASK [Agent Name]
- **What:** Description of task
- **Why:** Business rationale
- **Priority:** High/Medium/Low
- **Deadline:** YYYY-MM-DD
- **Success criteria:** How we know it's done
```

### Status Tracking
Agents update their status daily:
```
[Agent] Status: ✅ Active / ⏳ In Progress / ❌ Blocked
Progress: XX%
Next: ...
```
