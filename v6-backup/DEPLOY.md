# Deploy quán trendy café game lên Cloudflare Pages

# Cách 1: Dùng Cloudflare API (nhanh nhất)
# Cách 2: Dùng Netlify Drop (kéo thả)
# Cách 3: Upload lên GitHub repo thủ công trên điện thoại

# Hướng dẫn nhanh nhất cho người không có máy tính:
# 1. Mở https://pages.cloudflare.com
# 2. Sign up free
# 3. Create Pages → Direct Upload
# 4. Download worker.dev tool hoặc dùng CLI
# 5. Upload 3 file: index.html, styles.css, game.js

# Hoặc dùng Netlify:
# 1. Mở https://app.netlify.com/drop
# 2. Sign up bằng email
# 3. Kéo thả folder trendy-cafe-game vào
# 4. Xong! Có link ngay
