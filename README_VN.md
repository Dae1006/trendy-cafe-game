# 🎮 Hướng dẫn Deploy Quán Trendy Café — Từ điện thoại

## ✅ Cách 1: Netlify Drop (DỄ NHẤT — 2 phút)

1. Mở trình duyệt trên điện thoại
2. Vào **https://app.netlify.com/drop**
3. Sign up free bằng email (hoặc Google)
4. **Tải folder `trendy-cafe-game` về điện thoại**
5. **Kéo thả folder vào trang Netlify**
6. ✅ Xong! Bạn có link: `https://random-name.netlify.app/`
7. Chia sẻ link cho bạn bè chơi!

---

## ✅ Cách 2: GitHub Pages (3 phút)

1. Tạo GitHub account nếu chưa có: **https://github.com/signup**
2. Tạo repo mới: tên `trendy-cafe-game`
3. Upload 3 file:
   - `index.html`
   - `styles.css`
   - `game.js`
4. Vào **Settings → Pages**
5. Source → **Deploy from a branch** → chọn `main` → `/ (root)`
6. ✅ Xong! Link: `https://<username>.github.io/trendy-cafe-game/`

---

## 📱 Cách upload file từ điện thoại vào GitHub:

1. Mở **GitHub app** hoặc **github.com** trên điện thoại
2. Vào repo của bạn
3. Nhấn **Add file → Upload files**
4. Chọn 3 file: `index.html`, `styles.css`, `game.js`
5. Nhấn **Commit changes**
6. Vào Settings → Pages → deploy

---

## 🚀 Cách 3: Cloudflare Pages

1. Vào **https://pages.cloudflare.com**
2. Sign up free
3. **Create Pages → Direct Upload**
4. Download Cloudflare Pages CLI hoặc dùng web UI
5. Upload 3 file
6. ✅ Xong!

---
