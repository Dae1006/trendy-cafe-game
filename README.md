# ☕ Quán Trendy Café — Web Game

Quản lý quán cà phê, thu thập vật phẩm trendy, trang trí quán, và kiếm tiền!

## Features
- Multiple cafe locations (vỉa hè, TMall, sân vườn, rooftop, lễ hội...)
- Trend-driven events & collectible items
- Quest system → thu thập items đặc biệt
- Items: trang trí quán hoặc tạo món signature → tăng doanh thu
- Player marketplace (mua bán items)
- Rarity system: Common → Rare → Epic → Legendary
- Staff auto-serve system
- Upgrade & expansion

## Tech Stack
- HTML5 + CSS3 + Vanilla JS
- LocalStorage for save data
- Firebase (optional, for multiplayer)

## Quick Start
Mở file `index.html` trong trình duyệt.
