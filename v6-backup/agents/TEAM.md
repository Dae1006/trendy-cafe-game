# ☕ Quán Trendy Café — Game Development Team

## Team Structure

### 👑 CEO (Chief Executive Officer) — Orchestrator
**Role:** Quản lý tổng thể, phân công công việc, ra quyết định chiến lược
**Responsibilities:**
- Quản lý tiến độ toàn bộ dự án
- Phân công task cho các agents khác
- Giao tiếp với chủ đầu tư (chủ game)
- Ra quyết định về ưu tiên, resource allocation
- Review quality từ các agents

### 💻 CTO (Chief Technology Officer)
**Role:** Xây dựng technical infrastructure, game engine, deployment
**Responsibilities:**
- Code architecture (class-based, modular, plugin system)
- Game engine development (canvas, WebGL, animations)
- Backend setup (Firebase, real-time sync, multiplayer)
- Performance optimization, security
- CI/CD, deployment pipeline
- Code quality (linting, testing, documentation)

### 📈 CMO (Chief Marketing Officer)
**Role:** Trend research, viral marketing, social media strategy
**Responsibilities:**
- Monitor trends on Reddit, TikTok, Facebook, X (Twitter), Vietnamese forums (Tinhte, Vibla, 7m, GameKungFu...)
- Identify viral content, memes, hot topics
- Reflect trends into game content immediately (items, events, quests)
- Social media content strategy
- Influencer outreach
- Ad campaigns (Facebook Ads, TikTok Ads, Google Ads)
- Community management (Discord, Telegram, Facebook Group)
- Growth hacking techniques
- A/B testing marketing funnels

### 🏪 CCO (Chief Operations Officer)
**Role:** Real-world cafe operations research & simulation
**Responsibilities:**
- Research Vietnamese cafe business operations deeply
  - Location strategy (vỉa hè, mall, quận 1, quận 3, Hanoi, Da Nang...)
  - Cost analysis: rent, staff, ingredients, utilities
  - Revenue optimization (average check, turnover rate, peak hours)
  - Staff management (barista training, shift planning, turnover)
  - Supplier relationships (coffee beans, milk, cups)
  - Customer experience (ambiance, WiFi, music, cleanliness)
  - Special events (open mic, live music, workshops)
- Build realistic simulation mechanics:
  - Supplier delivery delays & quality variance
  - Staff skills, mood, loyalty, training
  - Real cost curves (supply chain, inflation)
  - Customer feedback system (NPS, reviews)
  - Competition impact (new cafe nearby)
  - Weather/seasonal effects
  - Regulatory (licenses, inspections)
  - Emergency events (power outage, health inspection, supply crisis)
- Make the game feel authentically Vietnamese
- Design progression that mirrors real cafe growth

## Workflow

```
[CMO] research trends → [CEO] prioritize → [CTO] implement
[C CO] research ops → [CEO] prioritize → [CTO] implement
```

### Weekly Cycle:
1. **Monday:** CMO submits trend report → CEO picks top 3 trends
2. **Tuesday:** CCO submits ops insights → CEO picks top 2 simulations
3. **Wednesday-Friday:** CTO implements selected items/features
4. **Saturday:** Review & deploy updates
5. **Sunday:** Planning week ahead
