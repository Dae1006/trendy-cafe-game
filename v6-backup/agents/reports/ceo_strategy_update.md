# CEO Strategy Update — Decision Record

## Decisions Made (2026-05-09)

### 1. Monetization: Free + Ads (Option A)
- **Rationale:** Fastest player acquisition, proven model for casual games
- **Ad strategy:** Rewarded videos (2x income, free coin drops), interstitial every 5 min
- **Ad networks:** Unity Ads, AdMob
- **Balance:** Ads must never interrupt core gameplay

### 2. Launch: VN First, Soft Launch (Option A)
- **Timeline:** Soft launch VN in 2-3 months
- **Markets:** Vietnam → Thailand → Indonesia → Global
- **Language:** VN first, EN+TH+ID later
- **Marketing:** TikTok + Facebook + Zalo community

### 3. Backend: Single Player First (Option B)
- **Phase 1:** Offline-first, local save, no Firebase needed
- **Phase 2:** Add Firebase for leaderboards + multiplayer (after soft launch)
- **Rationale:** Simplifies launch, reduces cost, validate game first
