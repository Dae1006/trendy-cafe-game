# ☕ QUÁN TRENDY CAFÉ — PIXEL ART EDITION

Game quản lý quán cà phê phong cách **Pixel Art Retro**! 🎮

## 🎨 Pixel Art Features

- **Font chữ pixel**: Press Start 2P (retro game font)
- **Border pixel**: Bo góc vuông, viền cứng 4px
- **Drop shadow**: Hiệu ứng đổ bóng pixel trên emoji
- **Grid overlay**: Lưới 4px trên background cho cảm giác retro
- **Pixel animations**: Bounce, shake, fade-in
- **Retro colors**: Gam màu game 8-bit cổ điển
- **Pixelated rendering**: `image-rendering: pixelated` cho smooth pixels
- **Button effects**: Press-in/press-out như game cartridge

## 🎮 Gameplay Features

- **5 vị trí quán**: Vỉa hè → Mall → Sân vườn → Rooftop → Lễ hội
- **10 món menu**: Cà phê đen, trà sữa, bubble tea, signature drinks...
- **13 trend items**: Meme chó, Áo MU, Vua Cà Phê, Omega Plus...
- **8 nhiệm vụ**: Quest system với rewards
- **10 upgrades**: Nâng cấp quán
- **Weather system**: Thời tiết ảnh hưởng doanh thu
- **Supplier system**: Quản lý nguyên liệu
- **Staff system**: Thuê và quản lý nhân viên
- **Save/Load**: Tự động lưu localStorage
